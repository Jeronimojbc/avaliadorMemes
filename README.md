# Aplicação de Gestão de Memes

Esta é uma aplicação web para gerenciar memes. Ela Permite que os usuários criem, visualizem, avaliem e excluam memes. Desenvolvida em PHP com um banco de dados MySQL.

## Requisitos

- PHP 8.0
- Servidor web (Apache)
- MySQL
